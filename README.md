# TGraphics

## Installation
> # Via `pip`
> ```
$ pip install tgraphics
```

> # Via Source from GitHub using `git`
> ```
$ git clone

## Usage
```py
from tgraphics import (
    Screen,
    Sprite,
    Vector, # Experimental, b/w vector
    Texture # Experimental, b/w texture (wrapper for vector)
)
```
